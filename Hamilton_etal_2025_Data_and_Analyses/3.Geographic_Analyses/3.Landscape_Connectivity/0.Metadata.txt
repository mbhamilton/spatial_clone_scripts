"1.connectivity_inter" contains all between population comparisons for cumulative current (landscape connectivity) from Circuitscape runs.

"2.connectivity_inter" contains all within populations, between patch, comparisons for cumulative current (landscape connectivity) from Circuitscape runs.

"3.PercDiff.xlsx" contains all percent difference comparisons for land classes found via connectivity analyses and general land classification areas.

"4.Inter" contains all files necessary to run Circuitscape for between-population comparisons in a server-sided HPC environment. Please contact Charli Minsavage-Davis (charli.mindav@gmail.com) for more information on this analysis!

"5.Inter" contains all files necessary to run Circuitscape for within-population, between-patch, comparisons in a server-sided HPC environment. Please contact Charli Minsavage-Davis (charli.mindav@gmail.com) for more information on this analysis!

You can find the Circuitscape user guide at "https://docs.circuitscape.org/Circuitscape.jl/latest/"
