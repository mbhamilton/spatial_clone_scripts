"1.LI_INEST_Results" contains all data and results for analyses run through the program INEST. These include analyses for the impacts of null alleles, estimates of inbreeding coefficients, and expected heterozygosity corrected for null alleles.

"2.Program_Files" contains all information and the files needed to run an INEST analysis.
